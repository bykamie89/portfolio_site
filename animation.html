Animation

I dette tema skulle vi udvikle et spil. Til udarbejdelsen af designet til vores spil blev vi introduceret til forskellige stiltyper, animations principper, skitseringsteknikker, idegeneringsteknikker mm. dette benyttede vi blandt andet til udarbejdelsen af vores papir prototyper der dannede grundlag for design af vores assets, ui-elementer, forgrund, evt mellemgrund og baggrunde. Til udvikling af vores figur og spil elementer benyttede vi os af programmet Adobe Illustrator hvor vi eksporterede alle filerne til .svg.

Til udviklingen af spillet skulle vi ud fra vores ide skabe et aktivitetsdiagram der viser hvordan flowet i spillet skal være og senere state machine diagram der tydeligt viser hvilke funktioner der skulle kodes og hvad de skulle indeholde. Dette diagram kunne vi overføre vil VS Code så det hele tiden var med til at skabe et overblik over udviklingen.

Vi har løbende gennem det meste af temaet fået viden omkring blandt andet JavaScript som vi skulle benytte til at sammensætte alle spillets funktioner, vi har lært om css animationer, implementering af lyde, forskellige metoder i javascript, events, betingelser, random genering mm. for at se sitet og afprøv spillet: klik her
